# What Is Coming Next

_Published: March 13, 2026_

The first release of HyprFlux is only the starting point.

Now that the project is public, the next phase is about refinement. The goal is not to add features for the sake of adding more. The goal is to improve the overall experience while keeping HyprFlux focused.

## Current priorities

The next updates are likely to focus on:

- improving installer clarity and reliability
- expanding documentation for customization
- refining theme consistency across components
- making config modules easier to understand
- polishing small workflow details that matter daily

## Documentation as part of the product

One important part of HyprFlux is the docs themselves. The documentation should help new users get started quickly, but it should also help existing users learn how the setup is structured and how to make changes without breaking the overall design.

That means future work will keep improving both the desktop and the writing around it.

## Building in public

Publishing updates like this is also part of the process. It creates a record of how HyprFlux grows and makes it easier for users to follow the direction of the project over time.

More notes, release posts, and project updates will follow as the setup evolves.
